- 문제의 '최소 몇 번 만에 빨간 구슬을 구멍을 통해 빼낼 수 있는지 출력한다.' 에서 최단거리를 찾야함을 알 수 있다. 따라서 BFS 탐색한다.
- 이 문제에서 이동 횟수를 카운트 할 때 중요한 것은 **Depth에 따라서 카운터를 증가**시켜야 한다는 것이다. 즉 현재 Queue 상태에서 모든 Item 들을 탐색하면서 각 Item 별 다음 단계로 진행 가능한 상태를 모두 저장한 후 카운터를 증가 시킨다.
- 공 이동 시 벽이나 구멍을 만날 때까지 진행된다. 방문 체크 시 공이 이동한 경로를 모두 저장하는 것이 아니다.(이동 경로를 모두 체크하게 되면 탐색하지 못 하는 경우가 발생한다.) *BFS에서 방문 체크는 각 탐색의 시작 위치를 방문 체크하여 탐색의 중복을 피해야한다.* (탐색 위치를 트리나 그래프의 노드라고 생각한다.)
